# 🎋 BambooLandTrees

> **"¡Tala árboles gigantes en un segundo, sin complicaciones y con total seguridad!"**

¡Bienvenido al repositorio oficial de **BambooLandTrees**! Este plugin de Minecraft está diseñado para llevar la recolección de madera a otro nivel. Olvídate de picar bloque por bloque; con nuestro sistema inteligente, talar un árbol es tan satisfactorio como debe ser.

---

### 🌟 ¿Por qué elegir BambooLandTrees?
* **Tala Inteligente:** Detecta el árbol completo, incluyendo estructuras masivas o complejas.
* **Seguridad Primero:** El plugin es "inteligente": si detecta que la madera forma parte de una construcción (porque no tiene hojas conectadas), ¡no hará nada! Tus casas están a salvo.
* **Balance Realista:**
    * Consume la durabilidad de tu hacha bloque por bloque.
    * Respeta encantamientos como *Irrompibilidad* (Unbreaking) y *Reparación* (Mending).
    * Obtienes exactamente los mismos *drops* (manzanas, retoños, madera) que talando manualmente.
* **Optimización Extrema:** Diseñado para ser ligero y no causar lag, incluso en servidores con muchos jugadores.
* **Totalmente Actualizado (1.21.11):** Soporte completo para el nuevo Roble Pálido, Bambú y la nueva **Hacha de Cobre**.

---

### ⚙️ Guía de Uso

#### Para Jugadores
Simplemente ten un hacha en tu mano principal y usa los comandos:
* `/BLTree on` — Activa la tala automática.
* `/BLTree off` — Desactiva la función (vuelve a la tala clásica).

#### Compatibilidad de Herramientas
El plugin reconoce todas las hachas oficiales:
`Madera`, `Piedra`, `Cobre`, `Hierro`, `Oro`, `Diamante` y `Netherite`.

---

### 🛠️ Información para Administradores

* **Versión del plugin:** 1.0.0
* **Compatibilidad:** Minecraft 1.21.11
* **Instalación:** El archivo compilado `.jar` se encuentra dentro de la carpeta `/target` de este repositorio.

#### Permisos
| Permiso | Descripción | Por defecto |
| :--- | :--- | :--- |
| `bambolandtrees.use` | Permite usar el comando /BLTree | Todos |
| `bambolandtrees.admin` | Permisos de configuración y gestión | Solo OP |

---

### 📋 ¿Qué soporta el plugin?
El sistema reconoce automáticamente:
* **Troncos:** Roble, Abeto, Abedul, Jungla, Acacia, Roble Oscuro, Manglar, Cerezo, Roble Pálido y Bloques de Bambú sólido.
* **Variantes:** Soporta todas las versiones de madera ("Wood"), troncos sin corteza ("Stripped Logs") y madera sin corteza ("Stripped Wood").
* **Follaje:** Elimina automáticamente las hojas (incluyendo Azaleas) sin consumir durabilidad adicional.
